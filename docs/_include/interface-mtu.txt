  {{ var5 }} {{ var6 }} mtu <mtu>

  Configure :abbr:`MTU (Maximum Transmission Unit)` on given `<interface>`. It
  is the size (in bytes) of the largest ethernet frame sent on this link.

  Example:

  .. code-block:: none

    set interfaces {{ var0 }} {{ var1 }} {{ var2 }} {{ var4 }} {{ var5 }} {{ var7 }} mtu 9000
